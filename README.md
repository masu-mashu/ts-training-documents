# GitBook

## 環境構築

- gitbook-cli インストール

```sh
$ npm install gitbook-cli -g
$ gitbook --version
```

## ローカルサーバー起動

- serve コマンド使用

```sh
$ gitbook serve
```

- http://localhost:4000 を開き確認
